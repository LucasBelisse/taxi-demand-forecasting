🚕 Previsão de Demanda de Táxis (Séries Temporais)

Este projeto foi desenvolvido para prever a quantidade de pedidos de táxi para a próxima hora em aeroportos, auxiliando na gestão de frota da Sweet Lift Taxi.

## 📊 Resultado Principal
- **Meta de Erro (REQM):** Abaixo de 48
- **Resultado Final:** **43.65** ✅

## 🛠️ Tecnologias Utilizadas
- **Python** (Pandas, Scikit-Learn, Statsmodels, Matplotlib).
- **Técnicas:** Engenharia de Recursos (Lags e Médias Móveis), Análise de Sazonalidade e Modelos de Regressão.
